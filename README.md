> **Unified Visual Customization Experience across Android, Wear OS, and Android TV.**

WatchTheme is a cross-platform visual ecosystem that allows users to synchronize themes, colors, and visual assets seamlessly across their entire device suite. Built with Flutter, it leverages a shared codebase to deliver a consistent and premium experience on mobile, wrist, and living room screens.

*   **Cross-Platform Synchronization**: Instant theme application across connected devices.
*   **Material You Integration**: Dynamic color extraction and application.
*   **Wear OS Companion**: Optimized layout for circular displays with rotary input support.
*   **Android TV Interface**: Leanback-inspired layout for large screens.
*   **Performance**: 90% code reuse with platform-specific adaptations where it counts.
